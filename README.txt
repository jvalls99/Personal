The programme filter.py intends to detect which samples have been excluded when it comes to performing sequencing analysis in Klebsiellas.
Several methods are included during these analysis such us QUAST,FASTP,TRIMME or PROKKA, so that after every step we filter our reads and
assemblies according to prior knowledge such us expected number of CDS or average length of the reads.

THe different input files must be provided as arguments of the .py programme.
